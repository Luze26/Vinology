Accueil

Parce qu'on aime le vin.

Introduction

Le site se base sur une ontologie créée via Protégé en utilisant OWL, RDF, RDFS
L'ontologie a pour but de rassembler les connaissances sur les vins de Savoie.


Fonctionnalités

Le site donne un accès rapide à nos vins, nos cépages et nos producteurs via les onglets du menu.
La plus part des tableaux de données peuvent être triés sur une colonne en cliquant sur son en-tête.
Chaque ressource de notre ontologie est également liée à une page web permettant d'afficher ses infos, permettant de naviguer au travers de l'ontologie sans s'arrêter.

Requêtez notre ontologie

Vous pouvez également directement requêter notre ontologie via le dernier onglet. La requête doit respecter la syntaxe SPARQL. Quelques exemples vous sont propoés en cliquant sur le menu déroulant en-dessous du champ de requête.
La liste des résultats est affichée en-dessous, deux afficahges sont proposées: "Traité" affichant de manière plus lisible les résultats liant les ressources aux pages web, ou "brut" pour l'affichage sans traitement.
Les liens sur les ressources renvoit sur une page d'information de la ressource, ou si la ressource est un individu de notre ontologie vers la page correspondante (vin, cépage, cru, ...).
